Alias Commands
#alias #alias <pattern> #alias <name> <expansion> Examples: #alias heal cast 'heal' %1 #alias prep stand;draw sword;cast 'armor' #alias sayall gossip %0 #alias mtell tell %1 [Tom says:] %r Variables: %0 = all text after the alias name %1..%9 = positional arguments %r = remaining text after the highest referenced positional variable
Bind Commands
#bind #bind <pattern> #bind <key> <expansion> Examples: #bind f2 flee #bind pageup scan #bind up north #bind numpad8 north Notes: f2-f12 and the listed navigation keys are supported. f1 is reserved and cannot be rebound. Binds use the same command pipeline as typed input.
